## Review

> 유클리드 호제법

최대공약수와 최소공배수를 구할 두 수 중 큰 수와 작은 수를 정한 뒤

작은 수로 큰 수를 나누어 나머지를 구한다.

if ( 나머지 == 0)

그때의 작은 수가 `최대공약수`이고,

원래 두 수를 곱한 값을 최대공약수로 나눈 값이 `최소공배수`이다.

else 

큰 수 = 작은 수

작은 수 = 나머지

나머지가 0이 될 때까지 반복

## Code
{% highlight cpp %}

#include<iostream>
using namespace std;

int main(){
    int a,b;
    int big, small;
    int mok, nmg, gcm, lcm;
    
    cin >> a >> b;
    if ( a >= b){
        big = a;
        small = b;
    }
    else{
        big = b;
        small = a;
    }
    while (1) {
        mok = big / small;
        nmg = big - mok * small;
        if (nmg == 0){
            gcm = small;
            lcm = a * b / gcm;
            cout << gcm << " " << lcm << endl;
            break;
        }
        big = small;
        small = nmg;
    }
}
